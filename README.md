## 📌 Sobre o Projeto

Este projeto foi desenvolvido como parte do meu aprendizado em análise de dados com Power BI. O objetivo foi construir um dashboard interativo de vendas, apresentando os principais KPIs e permitindo a comparação de desempenho entre diferentes períodos.

---

## 🎯 Objetivos

- ✅ Criar um dashboard interativo e intuitivo
- ✅ Apresentar os principais KPIs de vendas
- ✅ Comparar o desempenho entre períodos (mensal, trimestral, anual)

---

## 📈 KPIs Apresentados

| Indicador |

| Total de Vendas |

| Fidelidade |

## 🛠️ Ferramentas Utilizadas

## 🚀 Como Visualizar

1. Faça o download do arquivo `CHOCOLATEDATASET.pbix`
2. Abra com o **Power BI Desktop** (gratuito em [powerbi.microsoft.com](https://powerbi.microsoft.com))
3. Explore o dashboard interativo
